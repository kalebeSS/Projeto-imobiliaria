# Projeto Imobiliário

Este projeto tem como objetivo realizar uma análise de preços imobiliários em uma cidade hipotética.

## Estrutura do Projeto
- `data/`: Contém os dados brutos e gerados.
- `notebooks/`: Notebooks Jupyter para análise exploratória.
- `src/`: Código-fonte organizado em módulos.
- `README.md`: Documentação do projeto.

## Tecnologias Usadas
- Python 3.x
- Pandas
- Matplotlib / Seaborn
- Scikit-learn (opcional)

## Como Rodar

1. Clone o repositório:
    ```bash
    git clone https://github.com/username/projeto-imobiliaria.git
    cd projeto-imobiliaria
    ```

2. Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```

3. Execute o gerador de dados (se necessário):
    ```bash
    python data/imoveis_generator.py
    ```

4. Abra o notebook `EDA.ipynb` no Jupyter Notebook para visualizar a análise e as visualizações.
